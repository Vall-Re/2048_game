# 2048

**2048** is a popular puzzle game where the player moves tiles on a 4×4 grid, merging identical numbers.
The goal is to create a tile with the number **2048** by using logic and strategic thinking.

## Live Preview

## Technologies Used
-- **HTML** — page structure
-- **CSS** — styling and layout
-- **JavaScript** — game logic and interactivity

---

## Game Features
-- Merge tiles with identical numbers
-- Track your current score
-- Control the game with arrow keys
-- Start a new game anytime

## Getting Started
Running the Project Locally

1. Clone the repository:

-- git clone [GitHub repository](https://github.com/vall-re/2048_game.git)

2. Install dependencies (if any):

npm install
# or
yarn install

3. Start the game locally:

npm start
# or
yarn start
